# 🔍 GitHub Personality Analyzer

Анализирует GitHub-профиль и определяет тип разработчика на основе активности и языков программирования.

## 🚀 Возможности
- Анализ репозиториев GitHub
- Определение типа разработчика
- Подсчёт активности
- CLI-интерфейс с цветным выводом
- Юмористическое описание

## ▶️ Запуск
```bash
pip install -r requirements.txt
python analyzer.py
